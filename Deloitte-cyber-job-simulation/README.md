# Deloitte Australia Cyber Job Simulation

## Project Overview
This repository documents my completion of the Deloitte Australia Cyber Job Simulation on Forage. The simulation tasked me with acting as a cybersecurity consultant, investigating a data breach for a client and identifying the source of the leak. I analyzed web activity logs to find suspicious user behavior and provided actionable recommendations.

**Key Skills Practiced:**
*   Log Analysis
*   Incident Investigation
*   Web Security & Threat Identification
*   Professional Cybersecurity Reporting

## Simulation Task: Cybersecurity Investigation

**Goal:** To support a client in determining the source of a data breach by analyzing web activity logs.

### Part 1: The Incident
A client has reported a potential leak of private company information. I was brought in to investigate.

### Part 2: Log Analysis
The primary task was to analyze web server logs to identify the root cause. My approach was to:
1.  **Review Log Data:** Carefully examined the provided log files, looking for anomalous entries.
2.  **Identify Suspicious Activity:** Searched for patterns indicating a breach, such as:
    *   Unusual data access times (e.g., after hours).
    *   Data exfiltration patterns (large data transfers).
    *   Unauthorized access attempts.
3.  **Pinpoint the Source:** Correlated suspicious activities to a specific user account.

### Part 3: Findings and Recommendation
Based on the analysis, I produced a summary of findings and a recommendation to prevent future occurrences. This simulated the formal reporting process a Deloitte consultant would follow.

## Key Learnings & Takeaways

*   **How to Read Web Logs:** This simulation gave me practical experience in parsing and understanding web server logs to detect security incidents.
*   **Incident Investigation Process:** I learned how to approach a cybersecurity breach systematically, moving from detection to analysis and recommendation.
*   **Professional Communication:** The exercise emphasized the importance of clear, concise reporting for a client who may not have deep technical expertise.
