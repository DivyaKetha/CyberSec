# Deloitte Australia Cyber Job Simulation

## Project Overview
This repository documents my completion of the Deloitte Australia Cyber Job Simulation on Forage. The simulation tasked me with acting as a cybersecurity consultant, investigating a data breach for a client and identifying the source of the leak. I analyzed web activity logs to find suspicious user behavior and provided actionable recommendations.

**Key Skills Practiced:**
*   Log Analysis
*   Incident Investigation
*   Web Security & Threat Identification
*   Professional Cybersecurity Reporting
